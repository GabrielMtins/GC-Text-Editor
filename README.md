To compile just type it:

```
make  
```

![alt_image](https://i.imgur.com/9YgGcBR.png)  

## Commands

To enter command mode, press ESC and type one of these commands:  

```
    j <line> - jump to a line
    r <file> - read a file
    w <file> - write to a file
    q - quit program
```
